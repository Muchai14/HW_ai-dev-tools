# Django TODO App

This is a simple TODO application built with Django for Homework 1. It supports creating tasks, adding descriptions, setting due dates, marking tasks as done, and deleting tasks.

## How to Run
1. Install Django:
   pip install django

2. Go to the project folder (where manage.py is located):
   cd todo_project

3. Apply migrations:
   python manage.py migrate

4. Run the server:
   python manage.py runserver

5. Open in the browser:
   http://127.0.0.1:8000/

## Run Tests
python manage.py test
